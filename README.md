##  Featured Projects

###  Job Market Tracker — ELT Pipeline

Ingests job listings from Adzuna API with pagination, rate limiting & retries. Unified data model with automatic skill extraction, Snowflake loading, dbt ELT modeling with tests and doc generation, orchestrated end-to-end by Airflow.
